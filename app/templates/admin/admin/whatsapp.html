Access Token Expired - Action Required

Your WhatsApp access token has expired. You need to generate a new token from Meta's Developer Console.

Steps to Get a New Access Token:

  1. Go to https://developers.facebook.com/apps
  2. Select your WhatsApp Business App
  3. Navigate to WhatsApp → API Setup
  4. Find the "Temporary access token" section (or generate a System User token for permanent access)
  5. Copy the new access token
  6. Update WHATSAPP_ACCESS_TOKEN in your .env file
  7. Restart your Flask application
